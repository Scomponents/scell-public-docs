SCell Public API Interfaces Release Notes

Version 0.1.2 (June 2022)

####Changes

#####Renamed:

  • IScellUiControlSettings -> IUiControlSettings - it seems the SCell word is extra here;
  • IScellUiApi<TControl>.getController() -> getSettingsService() - to remove double meaning with Controller word.

#####Changed:

  • IFileWriter.save() - replace the return value to CompletableFuture<Boolean> to indicate save success

#####Added:

  • IUiSelectionManager - service to control cells selection:
    • setSelection(IRangeAddress) - possibility to set selection to a cells range;
    • getSelection() - returns CompletableFuture of IRangeAddress to have current selection coordinates;
    • setOnSelectionChange(BiConsumer<IRangeAddress, IRangeAddress> listener) - sets listener to the active cell chanded event
  • IScellUiApi<TControl>.getSelectionManager() - getting the IUiSelectionManager object.
  • IRangeAddress.toA1String() - returns user-readable cell address;
  • IRangeAddressBuilder - add possibility to set separate row or columns numbers. Convenient when building cell address from existing:
    • setTopLeft(int rowNumber, int columnNumber),
    • setBottomRight(int rowNumber, int columnNumber),
    • setTopRow(int rowNumber),
    • setLeftColumn(int columnNumber),
    • setBottomRow(int rowNumber),
    • setRightColumn(int columnNumber);
    • fromA1String(String source) - possibility to recognize string as A1-like cell address
  • ISpreadsheetApiProvider
    • IProductInfo getProductInfo() - returns service to get some build information
  • IProductInfo - service to get some build information
    • String getApiInterfacesVersion()- Current Public API Interfaces library version string
    • String getApiImplementationVersion() - Current Public API Implementation library version string
    • String getBuildNumber() - Current SCell ordinary build number
  • IFileWriter.getDefaultFileName() - default file name to save current document if it was newly created
