Adding SCell API to your Java project

SCell Public API supports only Maven projects.

To use the SCell Public API you should have the settings.xml for your local Maven repository folder (.m2) configured with paths and credentials to the SCalable Component's Nexus.

Now you can connect SCell API as dependency in your Maven project:

    <dependencies>
        ...
        <dependency>
            <groupId>com.intechcore</groupId>
            <artifactId>scell-api</artifactId>
            <version>0.2.0</version>
        </dependency>
        ...
    </dependencies>

Now you can use first access-point of the API - ScellApiProvider.
