Change Log

1.5.7-HF4589 (latest publish release)

Spreadsheet Comparison

- Compare two spreadsheets side by side with clear highlighting of added, removed, and changed content

Visual & Usability Enhancements

- Insert and view images directly within spreadsheet cells
- Apply gradient backgrounds to enhance cell styling
- Improved copy/paste operations with Excel and other spreadsheet tools
- Export spreadsheets to PDF with formatting and layout preserved
- Previously unsupported objects can now be edited at the data level, while structure and formatting remain read-only
- Chart data (values and ranges) can be modified; chart type and layout remain fixed
- Data in conditionally formatted cells is now editable, without changing formatting logic

Other Improvements

- Fix parsing some types of hyperlinks (hotfix)
