SCell Public API

Description

SCell Public API aims to embed the SCell component to the customer's application.

SCell Public API consists of two Java libraries:

  1. SCell Public API Interfaces - com.intechcore.scomponents.scell.scell-api. It is a top-level dependency aims to be opened and well-readable.
  2. SCell Public API Implementation - com.intechcore.scomponents.scell.api-impl. It contains implementations of the API interfaces.

The following scheme shows the SCell Public API Interfaces architecture.

There is the ScellApiProvider which aims to provide the ScellApiResolver. The ScellApiResolver is an IOC container aims to contain the SpreadsheetApiProvider (core part), ScellUiFxApiProvider (UI part) and other API interfaces implementation.

The Workbook class is a core part entry point. It includes a set of Worksheet classes which manage WorksheetContent, WorksheetLayout, WorksheetService of itself.

The WorksheetContent manages content of cells in different points of view like hyperlinks, comments, styles and cells values content such as string, numeric, boolean etc…

The WorksheetLayout helps to work with rows and columns.

The WorksheetService does complex worksheet actions.

The ScellUiApi class is an UI entry point. It creates a new SCellUiContentManager instance to manage UI part of the SСell component.

For more information about how to start working see the Quick Start.
