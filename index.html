Description

SCell is a Java-based software component. You can use it in your application to work with spreadsheets. This includes importing xlsx files, editing their contents, formatting them.

It can be used as part of your Java application or as a standalone application, just like you use MS Excel or Google Sheets. This provides spreadsheet editing interface with most of its features.

SCell is developed on Java 8 and Java 11 is coming soon.

The SCell application can be embedded to your application by the SCell Public API. It was developed with deep understanding of our customers needs. Easy to read and use. An ever-expanding list of features.
