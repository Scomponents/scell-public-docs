SCell Public API Implementation Release Notes

Version 0.1.1-RC1 (May 19th, 2022)

It is the first release of SCell Public API Interfaces.

This release provides implementations of the following interfaces of the Public API:

  • IScellApiResolver - all the methods;
  • IScellUiFxApiProvider:
    • IScellUiApi<Node> create(CompletableFuture<IWorkbook> coreApi);
  • IScellUiApi<Node>:
    • Node createControl();
  • IUiContentManager:
    • CompletableFuture<Void> load(File file);
    • CompletableFuture<Void> clear();
  • IWorksheet:
    • int getId();
    • String getName();
    • boolean isHidden();
    • IWorksheetContent getContent();
  • IWorkbook:
    • String getName();
    • Optional<IWorksheet> find(int id);
    • Optional<IWorksheet> find(String name);
    • Stream<IWorksheet> all();
  • ISpreadsheetApiProvider - all the methods
  • IStyleService:
    • CompletableFuture<Void> clear(IRangeAddress range);
  • IWorksheetContent:
    • CompletableFuture<Void> inputContent(IRangeAddress range, String input);
    • IStyleContent getStyleContent();
  • IStyleContent:
    • IStyleService getStyleService();
  • IRangeAddress - all the methods;
  • IRangeAddressBuilder - all the methods;
